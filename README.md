# ethical-interrogation-game
A command-line interrogation game where you are questioned about a crime you know nothing about.

The system evaluates your responses as cooperative, evasive, or uncooperative, and adapts its tone and pressure accordingly. There are no objectively correct answers. The system judges you based only on how your behavior is interpreted.

## Concept
This game is an "unessay" based on the theory of what an ethical game looks like from "The Ethics of Computer Games" by Miguel Sicart. It explores how game systems do not simply respond to player choices, but interpret and shape them. The player acts as a moral agent, but the system ultimately determines meaning and outcome through its own rules and limitations. The goal of this game is to create situations where players can think ethically while playing, and to highlight how the limitations of a game's system can shape those situations.

## Instructions:
1) Clone the repository or download game.py
2) Open a terminal in the project folder
3) Run: python game.py
